샤이니컬러즈 커뮤 목록

위키 소스
Description
* 읽은 / 읽지 않은 커뮤를 기록할 수 있는 웹사이트입니다.
* 첫 이미지/아이콘 로딩에 시간이 필요할 수 있습니다. 10초 가량 기다려주세요.
* 창을 열 때는 클릭, 닫을 때는 Esc키를 사용합니다.
* Save Current State 버튼을 클릭하여 생성되는 URL로 현재 선택 상태를 저장할 수 있습니다.
* Save 버튼과 Load 버튼 사이의 칸에 해당 URL을 입력하고 Load Current State 버튼을 클릭하면 상태를 불러올 수 있습니다.
* 각종 링크는 위키 내의 대응하는 페이지로 이동합니다.

* Q: 링크에 페이지가 없어요. A: 위키 내에 대응하는 페이지가 없습니다.
* Q: 한글 제목으로 번역해주세요, 커뮤 링크를 유튜브 등으로 달아주세요. A: DB를 직접 업데이트해주시면 반영 가능합니다.
* Q: 최신 카드가 없어요. A: 210830 카드까지 업로드 완료하였습니다. 제작자가 수동 업데이트해야해서 업데이트가 늦어질 수 있습니다.
* Q: UI 예쁘게 만들어주세요, 사이트가 느려요, 기능 추가해주세요, ... A: 시간 나면 할게요.
* Q: 페이지가 이상해요. A: 하단의 개발자 이메일/깃헙 등으로 연락 바랍니다.
